# 🚀 Amazon Delivery Time Prediction

This project analyzes Amazon delivery data to understand the key factors that influence delivery time.  
We performed **EDA, geoanalysis, feature engineering, regression and classification models, and business insights**.

**Main results:**
- Random Forest Regressor achieved the best performance (MAE ~18 min, R² = 0.81).  
- Random Forest Classifier outperformed Logistic Regression (Accuracy ~78%).  
- Business insights revealed **Semi-Urban areas and categories like Sports/Kitchen/Cosmetics** as bottlenecks, while **Grocery** deliveries were unusually fast.  

📌 This project demonstrates how ML and analytics can be applied to logistics optimization.
